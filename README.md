### <img src="https://img.shields.io/badge/01-e94560?style=for-the-badge"/> Apple Stores | 1M+ Rows | Advanced

<table>
<tr>
<td width="65%">

**1,000,000+ rows** of Apple retail sales data across **5 interconnected tables**

Analyzes Apple's retail ecosystem — stores, products, sales transactions, and warranty claims. Tackles **20 advanced queries** designed for portfolio-level demonstrations.

<br/>

**Schema**
```
stores ── categories ── products ── sales ── warranty_claims
```

<br/>

**Key Techniques**

| Technique | Application |
|-----------|-------------|
| `LAG() OVER(PARTITION BY ...)` | YoY and MoM sales comparisons |
| `RANK()` / `DENSE_RANK()` | Product ranking within categories |
| `CORR()` | Warranty claim correlation analysis |
| `TRY_DIVIDE()` | Safe division handling |
| `SUM() OVER(...)` | Cumulative revenue tracking |

</td>
<td width="35%">

```sql
-- YoY Growth by Store & Product
WITH yearly_sales AS (
  SELECT
    store_id,
    product_id,
    YEAR(sale_date) AS yr,
    SUM(quantity) AS total_qty,
    LAG(SUM(quantity))
      OVER(
        PARTITION BY store_id,
                     product_id
        ORDER BY YEAR(sale_date)
      ) AS prev_yr
  FROM sales
  GROUP BY store_id,
           product_id,
           YEAR(sale_date)
)
SELECT *,
  ROUND(
    (total_qty - prev_yr)
    * 100.0 / prev_yr, 2
  ) AS growth_pct
FROM yearly_sales
WHERE prev_yr IS NOT NULL
ORDER BY growth_pct DESC;
```

</td>
</tr>
</table>

## <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Animated-Fluent-Emojis/master/Emojis/Objects/Rocket.png" alt="Rocket" width="28" /> Getting Started

<br/>

```bash
# Clone the repository
git clone https://github.com/rishigundla/sql-projects.git
cd sql-projects
```

```
sql-projects/
├── 100 Days of SQL/           # 75+ daily challenges + all_solutions.sql
├── Amazon SQL Projects/       # PostgreSQL | 7 tables | 19 queries
├── Apple Stores SQL Project/  # 1M+ rows | 5 tables | 20 queries
├── Target Store SQL Project/  # Databricks | Delta Lake | 12 queries
├── Walmart SQL Queries.sql    # 45 stores | Kaggle dataset
├── Covid Data India.sql       # National pandemic analysis
├── Marketing Insights/        # Campaign effectiveness
├── Pub Pricing Analysis/      # Beverage pricing chain
├── Finance Analysis/          # Banking analytics
├── eSports Tournament/        # Competitive gaming
├── Customer Insights/         # Retail segmentation
├── Steve's Car Showroom/      # Automotive sales
└── Healthcare Analysis/       # Clinical data
```

> **Prerequisites:** Any SQL client (MySQL Workbench, pgAdmin, DBeaver, VS Code) — Databricks workspace for the Target project.

Each project folder includes the SQL query file, datasets (where applicable), and a project-specific README with schema details.
